# PROJECT_CONTEXT — refacer

## Идентичность
`refacer` — внешний/апстрим-стиль инструмент AI-визуалов (multi-face deepfake / face-swap,
основан на xaviviro/refacer; использует Insightface). Это форк кодового проекта.

## Назначение в рамках workspace
- место для работы с кодом и документацией этого визуального инструмента;
- доработка docs, аккуратные локальные правки кода и подготовка инструкций — по запросу.

## Границы source of truth
- Source of truth для самого проекта — его файлы в этом репозитории (код, README, `.ai/*`).
- Этот репозиторий НЕ является источником истины для инфраструктуры, бизнеса или личной памяти.

## Чего здесь НЕ делают
- НЕ ведут бизнес-работу Pitaya (это `pitaya`);
- НЕ хранят личную память/медицину (это `Obsidian`);
- НЕ делают инфраструктурный routing / общую память (это `Dmitrii`);
- НЕ настраивают и не редактируют OpenClaw и CloudMem (это runtime-системы, не редактируемые проекты);
- НЕ ведут линию Agent Supervisor.
