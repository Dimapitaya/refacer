# AGENT_ROUTING — refacer

Какой агент для чего подходит (любое действие — только после явного запроса Дмитрия):

- **Claude Code / Cloud Code** — понимание репозитория, аккуратная документация,
  локальные правки кода после явного запроса.
- **Codex** — точечные патчи кода и тесты после явного запроса.
- **Antigravity** — IDE-навигация, планирование рефакторинга после явного запроса.
- **Hermes Agent** — оркестрация, статусы, чек-листы после явного запроса.

## Общие правила для всех агентов
- Оставаться внутри `refacer`, если явно не сказано иное.
- НЕ трогать runtime-системы (OpenClaw, CloudMem, gateway, сервисы) и посторонние репозитории.
- Если задача относится к другому репозиторию (Dmitrii / pitaya / Obsidian / Agent Supervisor /
  deprecated `telegram-task-memory-bot`) — остановиться и сообщить, не выполнять здесь.
- Не запускать refacer, не ставить зависимости, не качать модели, не обрабатывать изображения
  без явного разрешения.
