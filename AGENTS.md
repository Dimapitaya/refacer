# AGENTS.md — refacer

## Что это за репозиторий
Это `refacer` — рабочее пространство AI-визуалов / внешний форк (deepfake / face-swap инструмент,
апстрим: xaviviro/refacer). Это отдельный кодовый проект, НЕ память и НЕ инфраструктура.

## Не путать с другими репозиториями
`refacer` НЕ является и не должен смешиваться с:
- **Dmitrii** (AI-инфраструктура / память / routing, source of truth);
- **pitaya** (бизнес-память Pitaya);
- **Obsidian** (личная память);
- **Agent Supervisor** / `agent-supervisor-mvp` (отдельная параллельная линия);
- **OpenClaw** (runtime / оркестрация / Telegram-агенты);
- **CloudMem / Claude-Mem** (локальный слой памяти/кэша);
- **telegram-task-memory-bot** (DEPRECATED / legacy).

Не переносить сюда задачи, данные или контекст из этих репозиториев и не редактировать их отсюда.
